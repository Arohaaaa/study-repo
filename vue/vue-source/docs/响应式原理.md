# 响应式原理

## 核心原理

- 针对对象，主要使用 Object.defineProperty 进行实现

## 对象劫持

```js
function defineReactive(data, key, value) {
  // 如果值也是个对象，则继续往下劫持
  observe(value);
  Object.defineProperty(data, key, {
    get() {
      return value;
    },
    set(newValue) {
      if (newValue === value) {
        return;
      }
      // 设置的新值也是个对象时，继续往下劫持
      observe(value);
      value = newValue;
    },
  });
}
```
